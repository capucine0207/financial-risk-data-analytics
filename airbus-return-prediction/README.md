# Airbus Return Prediction

## Objective
Develop and compare statistical and machine learning models to predict weekly Airbus stock returns in a structured R environment.

## Context
This project was conducted as part of a group assignment focused on forecasting financial returns and selecting the best-performing model based on both statistical and qualitative criteria.

## Approach
- Selection of financial and economic explanatory variables  
- Data preprocessing and feature engineering in R  
- Train/test split for out-of-sample evaluation  
- Implementation of multiple models (OLS, Lasso, Random Forest, Boosting) in R  

## Model Evaluation
- Comparison using RMSE, MAE, and R²  
- Analysis of model robustness and generalization  
- Detection of overfitting and trade-off between complexity and interpretability  

## Key Insights
- Identification of key drivers of Airbus returns  
- Selection of the best-performing model based on statistical and qualitative criteria  

## Skills Demonstrated
- Financial time series analysis  
- Model comparison and validation  
- Data preprocessing in R  
- Critical evaluation of models
